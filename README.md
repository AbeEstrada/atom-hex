# Hex Viewer for [Atom](http://atom.io)

View the hex dump of any file

## Install

    apm install hex

If you want a shortcut, you can open your `keymap.cson` file and add this:

    '.editor:not(.mini)':
      'shift-ctrl-h': 'hex:view'
